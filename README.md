# Facebook Stream Bot 🚀

بوت واتساب احترافي مبني باستخدام مكتبة `@whiskeysockets/baileys` للتحكم في البث المباشر على فيسبوك (Facebook Live) باستخدام FFmpeg.

## 🌟 المميزات
- دعم تسجيل الدخول عبر **QR Code** أو **Pairing Code**.
- نظام أوامر احترافي (Command Handler).
- إدارة كاملة لعمليات FFmpeg (بدء، إيقاف، حالة).
- قاعدة بيانات JSON محلية لحفظ الإعدادات.
- إشعارات فورية عند توقف البث فجأة.
- حماية ضد الاستخدام غير المصرح به.

## 🛠️ المتطلبات
- Node.js 22 أو أحدث.
- FFmpeg مثبت على نظام التشغيل.

## 🚀 التثبيت

1. قم بتحميل ملف المشروع وفك الضغط.
2. افتح المجلد في Terminal.
3. قم بتثبيت الحزم المطلوبة:
   ```bash
   npm install
   ```
4. قم بتهيئة الإعدادات:
   - قم بتغيير اسم الملف `.env.example` إلى `.env`.
   - أضف رقم هاتفك (المالك) في ملف `.env`.

## ⚙️ تشغيل البوت
لتشغيل البوت:
```bash
npm start
```

### تسجيل الدخول:
- **QR Code**: سيظهر الكود في Terminal، قم بمسحه من تطبيق واتساب.
- **Pairing Code**: إذا قمت بتفعيل `USE_PAIRING_CODE=true` في ملف `.env` ووضعت رقم الهاتف، سيظهر لك كود مكون من 8 أرقام لإدخاله في واتساب.

## 📱 الأوامر
| الأمر | الوصف |
|-------|-------|
| `.setfb KEY` | حفظ مفتاح البث (Stream Key) |
| `.fbstream URL` | بدء البث من رابط M3U8 |
| `.stopfb` | إيقاف البث المباشر |
| `.fbstatus` | عرض حالة البث ووقت التشغيل |
| `.setrtmp URL` | تغيير رابط RTMP الافتراضي |
| `.help` | عرض قائمة الأوامر |

## 📁 هيكل المشروع
- `index.js`: الملف الرئيسي لتشغيل البوت.
- `config.js`: ملف الإعدادات العامة.
- `ffmpeg.js`: وحدة إدارة عمليات البث.
- `storage.js`: وحدة إدارة قاعدة البيانات.
- `commands/`: مجلد يحتوي على جميع الأوامر.
- `sessions/`: مجلد لحفظ بيانات تسجيل الدخول.

## ⚠️ ملاحظة هامة
تأكد من أن رابط M3U8 يعمل بشكل صحيح قبل بدء البث، وتأكد من أن خادمك يحتوي على موارد كافية (CPU/RAM) لمعالجة الفيديو عبر FFmpeg.
